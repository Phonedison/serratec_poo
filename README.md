# Exercícios Práticos

### Visão Geral

Este repositório é dedicado à resolução de uma série de exercícios práticos propostos durante o curso. O intuito é demonstrar a evolução contínua da **lógica de programação** e a **capacidade técnica** na resolução de problemas individuais.

---

<img align="left" src="https://user-images.githubusercontent.com/74038190/216649426-0c2ee152-84d8-4707-85c4-27a378d2f78a.gif" width="50">

### Metodologia e Entrega

Os exercícios são disponibilizados progressivamente e atualizados diariamente. A entrega e avaliação baseiam-se em:

- **Versionamento:** Uso do Git / GitHub para demonstrar o progresso.
- **Granularidade:** Commits frequentes para cada etapa lógica ou exercício finalizado.
- **Originalidade:** Implementações individuais, seguindo as políticas de integridade do curso.

---

### Tecnologias Utilizadas

- **Java**, linguagem Principal
- Git para controle de versão
